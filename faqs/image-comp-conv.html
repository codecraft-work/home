Frequently Asked Questions

1. What encoders are used in the app?

The app currently supports two image encoders:

  • JPEG: A popular image format that balances quality and file size, making it ideal for web use and sharing.
  • PNG: A lossless format that preserves image quality without compression artifacts, best suited for images requiring transparency.
  • WEBP (Beta Version): A modern image format that provides better compression efficiency while maintaining high quality. It helps reduce file size even more compared to JPEG & PNG.

We are continuously working on improving WEBP support for better performance and accuracy.

2. What features does this app have?

The app offers the following features:

  • Image Compressor: Reduces the size of image files without compromising quality.
  • Image Converter: Converts images between different formats to meet your needs.
3. How do I compress a image?

To compress a image using the app, follow these steps:

  1. Open the Image Compressor - Converter app on your device.
  2. Tap on any feature & Select Images that you want to compress.
  3. Adjust the compression settings, including image resolution (height/width), rotation, and output format.
  4. Tap on Compress Image and wait for the process to complete.
  5. Once done, the compressed image is automatically saved or you can share it directly.

The app ensures a balance between image quality and file size, making it an efficient solution for managing large images.

4. What image formats are supported?

The app supports a variety of image formats, making it easier for users to compress and convert images without compatibility issues. Here are the formats you can work with:

Commonly Used Formats:

  • JPG/JPEG: The most widely used format, ideal for web and photography.
  • PNG: Supports transparency, commonly used for graphics and web design.
  • WEBP: A modern format with better compression and quality.
  • GIF: Supports animation, often used for short clips and memes.

Other Supported Formats:

  • BMP: Uncompressed format with high quality but larger file size.
  • TIFF: Used for high-quality images and professional editing.
  • HEIC: A newer format with better compression, mainly used by Apple devices.
  • HEIF: A high-efficiency format that offers better quality at smaller file sizes.

You can easily convert images between these formats based on your needs. If you're unsure which format to use, PNG is generally the best choice for compatibility and quality.

5. Does the app reduce image quality?

By default, the app tries to maintain the best possible quality while reducing the file size. However, you have full control over compression settings. You can choose from different compression levels:

  • High Quality: Minimal compression, large file size, best quality.
  • Balanced: Moderate compression, good quality, reduced size.
  • Small File Size: Maximum compression, reduced quality, very small file size.

This allows you to customize the output based on your storage needs and preference.

The app is free to download and use, with basic features available at no cost. However, there are some premium features that we provide for free to use, such as:

  • Ad-free experience.
  • Higher resolution output.
  • Faster image compression speed.
  • Batch processing for multiple images at once.

You can use the free version for basic compression and conversion, and also upgrade to the premium version for advanced features.
